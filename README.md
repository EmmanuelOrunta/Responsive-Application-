# Responsive Application Development (ITEC634)

## Course Overview

This repository contains the laboratory projects completed for **ITEC634 – Responsive Application Development** at Australian Catholic University (ACU).
Throughout the semester, I developed responsive web, mobile, and cross-platform applications using ASP.NET Core, Android Studio, Kotlin, Jetpack Compose, AJAX, HTMX, and .NET MAUI.
The course provided practical experience in modern application development, covering web technologies, mobile development, asynchronous communication, responsive design, and API integration.

## Laboratory Projects

| Lab | Project | Description |
|------|----------|-------------|
| 1 | ASP.NET Hello World Application | Installed the .NET SDK and built a simple ASP.NET web application. |
| 2 | ASP.NET Web Form Application | Developed a Razor Pages application that processes user input using HTTP POST requests. |
| 3 | Multi-Page Razor Pages Application | Created a responsive multi-page web application with navigation, routing, and Bootstrap styling. |
| 4 | AJAX & HTMX Application | Implemented asynchronous communication using Fetch API and HTMX without full page reloads. |
| 5 | Android TODO Application | Built a native Android task management application using Kotlin and Jetpack Compose. |
| 6 | Cross-Platform Weather Application | Developed a .NET MAUI application that retrieves live weather data from the OpenWeatherMap API. |
