# Responsive Application Development (ITEC634)

## Course Overview

This repository contains the laboratory projects completed for **ITEC634 – Responsive Application Development** at Australian Catholic University (ACU).
Throughout the semester, I developed responsive web, mobile, and cross-platform applications using ASP.NET Core, Android Studio, Kotlin, Jetpack Compose, AJAX, HTMX, and .NET MAUI.
The course provided practical experience in modern application development, covering web technologies, mobile development, asynchronous communication, responsive design, and API integration.
